# Poultry Farm Management System - Frontend

Modern, Responsive Dashboard for Poultry Farm Management. Full Uzbek language support!

## Tech Stack
- **React.js + Vite**
- **Tailwind CSS**
- **Recharts** (Visual analytics)
- **Lucide Icons**
- **Axios**
- **React Toastify**

## Key Features
- **AI Advice Panel**: Smart recommendations based on real farm data.
- **Visual Analytics**: Interactive charts for eggs, sales, and expenses.
- **Full CRUD**: Manage flocks, production, sales, health, and expenses.
- **Reporting**: Export reports to CSV and printable layouts.
- **Responsive Design**: Works perfectly on mobile and desktop.

## Setup
1. `npm install`
2. Create `.env` file:
   ```
   VITE_API_URL=http://localhost:5000
   ```
3. `npm run dev`

## Deployment
Optimized for **Vercel** or Netlify.
Includes `vercel.json` for SPA routing support.
